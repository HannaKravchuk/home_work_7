Normalize CSS

Як на мене normalize css лише трохи підправив стилі які були за замовчування в браyзерах. Зробив стилі більш-менш схожими в різних браузерах. Зберіглись стандартні відступи. Від начебто зкорегував відмінності між браузерами.

Normalize CSS 2

Як на мене використання normalize css є краще. Він корегує відмінності між браузерами та не видаляє корисні стилі. Елементи зберігають свої стандарні властивості в оформденні. Та набагато простіше завадавати свої стилі. Як на мене nomalixe css є набагато практичнішим та зручнішим у використанні.
