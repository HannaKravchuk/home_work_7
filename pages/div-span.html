Experiments with div and span

What is the div element?

The div element in HTML is a container used to group tpgether other elements. It doesn't add any special styling or meaning by itelf - it is just a basic box or section. You can style it with css to change its size, color, and position, or use it to organize your page layout. It is commonly used to wrap sections of website, like headers, footers, or main content areas. If you dont't set a width for a div element, it will automatically take up the full width available in its container, this means it will stretch to fill the width of its parent element (like the body or another container), as much space as it's allowed.

Div is a block-level element

Div is a block-level element. This means it takes up the full width available and starts on a new line, pushing other elements below it. It is used to group larger sections od content, like paragraphs, images, or other blocks. You can set a specific width and heigth for a div element. Here's example in this block

Span is an inline element. It only takes up as much space as needed and does not start on a new line. It is used to style or control smaller pices of text or content within a line, like a word or a phrase inside a paragraph.

The div is a container thaat holds a block of text, and the span is used to apply special styling - like a different color or bolf font - to just a small part of that text. The span doesn't create a new line or separate block; it simply lets you style a part of the text differently within the same line.
